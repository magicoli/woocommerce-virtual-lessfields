== Description ==

Remove address fields on WooCommerce checkout page for orders with only virtual products.

Ignores products marked as domain names with [WooCommerce Domain Names](https://magiiic.com/wordpress/plugins/wc-domain-names/) plugin.

== Installation ==

* Just install and activate, no setup
